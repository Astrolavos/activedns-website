Purpose of the Active DNS Project

In order to facilitate research around DNS, this project actively submits iterative and recursive DNS lookups for global TLDs (e.g., COM, NAME, NET, ORG, BIZ) and lists of domain names (i.e., Tranco Top 1M, Public Blocklists) on a daily basis. Active DNS has been running since 2016. This data is freely available to the academic research community for incorporation into new or existing work.

For more information please review the RAID 2016 publication Enabling Network Security Through Active DNS Datasets (cite).
